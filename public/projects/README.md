# Project Media Files

Place your project images and videos in this directory.

## File Structure

For each project, add media files following this naming convention:

- `hive-app.png` - Hive iOS app screenshot
- `hive-web.png` - Hive web platform screenshot
- `prepcenter-dashboard.png` - PrepCenter dashboard
- `prepcenter-study.png` - PrepCenter study modes
- `gathr-app.png` - Gathr iOS app screenshot
- `gathr-demo.mp4` - Gathr demo video
- `internify-dashboard.png` - Internify dashboard
- `internify-search.png` - Internify search interface
- `marketatomy-dashboard.png` - MarketAtomy dashboard
- `marketatomy-report.png` - Sample report
- `cfc-website.png` - CodingForCharity website
- `cfc-projects.png` - CodingForCharity projects showcase

## Supported Formats

- **Images**: PNG, JPG, JPEG, WebP
- **Videos**: MP4, WebM

## Adding New Media

1. Add your image/video files to this directory
2. Update the `media` array in `data/projects.ts` with the correct file paths
3. The paths should start with `/projects/` (e.g., `/projects/hive-app.png`)
